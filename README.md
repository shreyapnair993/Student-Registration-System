Student Registration System :-

A simple and elegant student registration system built with HTML, CSS, and JavaScript. Users can register students (Name, ID, Email, Contact), edit or delete records, and data is saved in local Storage to survive page refreshes. The system features a responsive design with a hamburger menu, form validation (letters-only names, numeric IDs, valid email, 10-digit contact), and a dynamic scrollbar when records exceed five.

Files: index.html (home + form), records.html (student table), style.css, script.js, logo.png, README.md
